相等运算符

2022-05-18 | 文档 | 370 字 | 1 分钟

==

不同类型的值用相等运算符比较后的结果
类型(x) 类型(y) 结果
null undefined true
undefined null true
数 字符串 x == toNumber(y)
字符串 数 toNumber(x) == y
布尔值 任何类型 toNumber(x) == y
任何类型 布尔值 x == toNumber(y)
字符串或数 对象 x == toPrimitive(y)
对象 字符串或数 toPrimitive(x) == y

如果x和y类型相同，JavaScript会用equals方法比较这两个值或对象。
没有列在这个表格中的其他情况都会返回false。

toNumber方法对不同类型返回的结果
值类型 结果
undefined NaN
null +0
布尔值 如果是true，返回1；如果是false，返回+0
数 数对应的值
验证

原因：

  1. 首先，布尔值会被toNumber方法转成数，因此得到packt == 0;
  2. 其次，用toNumber方法转换字符串值。因为字符串中包含字母，所以会被转成NaN，表达式就变成了NaN == 0，结果就是false。

===

如果比较的两个值类型不同，比较的结果就是false。如果比较的两个值类型相同，结果会根据下表判断：

类型(x) 值 结果
数 x和y的值相等(但不是NaN) true
字符串 x和y是相同的字符 true
布尔值 x和y都是true或false true
对象 x和y引用同一个对象 true
